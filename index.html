Olá, eu sou o Pablo — mas pode me chamar de Ryc.

Tenho 24 anos, nasci e cresci em São Luís, no Maranhão, onde moro até hoje. Atualmente curso Análise e Desenvolvimento de Sistemas (ADS) na Uninter, ainda nos semestres iniciais. Meu objetivo é seguir carreira como desenvolvedor back-end Java, e este site é justamente meu primeiro passo nessa direção: meu portfólio pessoal, feito do zero em HTML, CSS e JavaScript puros.

Fora da programação, sou apaixonado por Fórmula 1 e automobilismo em geral — acompanho as corridas sempre que posso. Também gosto de jogar de vez em quando, nos momentos de folga.

Estou no início da minha jornada como programador, mas com bastante motivação para aprender, evoluir e, no futuro, trabalhar com tecnologia de forma profissional — inclusive buscando oportunidades internacionais ao longo do caminho.

Java

Cofrinho Multimoeda (POO)

Sistema de console que simula um "cofrinho" capaz de guardar valores em Real, Dólar e Euro. Trabalho da disciplina de Programação Orientada a Objetos, aplicando herança, classes abstratas, polimorfismo e sobrescrita de equals() / hashCode().

  • Classes Moeda (abstrata), Real, Dolar, Euro e Cofrinho
  • Menu interativo via console (Scanner)

Python

Lógica de Programação e Algoritmos

Conjunto de exercícios práticos: um programa que calcula o valor mensal de um plano de saúde com base em faixas de idade, e um sistema de pedidos para uma pizzaria com validação de entradas, acumulador de valores e laços de repetição.

  • Estruturas if / elif / else
  • Laços while, break e continue

Banco de Dados Relacional
