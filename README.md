# Portfólio — Pablo Rycardo

Portfólio pessoal desenvolvido para a Atividade Prática da disciplina
**Fundamentos da Programação Web** (Uninter — Curso de Análise e
Desenvolvimento de Sistemas).

🔗 **Site publicado:** https://doudryc.github.io/portfolio-pablo-rycardo/

## Sobre o projeto

Site responsivo de página única (single page), construído **sem
frameworks**, usando apenas:

- **HTML5**
- **CSS3** (puro, com variáveis CSS para os temas claro/escuro)
- **JavaScript** (puro / vanilla)

## Estrutura do projeto

```
.
├── index.html        # Estrutura principal e conteúdo do site
├── style.css         # Estilos, layout responsivo e temas claro/escuro
├── script.js         # Tema, menu responsivo, terminal animado e validação do formulário
├── arquivos/         # Arquivos das entregas mostradas na seção Portfólio
└── README.md
```

## Funcionalidades

- **Tema claro / escuro**, com preferência salva no navegador.
- **Menu responsivo** (estilo "explorador de arquivos"), que se adapta a
  dispositivos móveis.
- **Destaque automático** da seção atual no menu conforme a rolagem da página.
- **Terminal animado** com efeito de digitação na seção "Sobre mim".
- **Formulário de contato** com validação em JavaScript (campos obrigatórios
  e formato de e-mail) e simulação de envio com mensagem de confirmação.
